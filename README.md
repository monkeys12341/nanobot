### Step 2: Install nanobot

The installation steps depend on the file you download:

- **Windows (.exe)**:
  1. Double-click the downloaded `.exe` file.
  2. Follow the installer instructions by clicking "Next" or "Install".
  3. Choose the folder where nanobot will be installed or leave it as default.
  4. Finish the installation by clicking "Finish".

- **Windows (.zip)**:
  1. Right-click the `.zip` file and select "Extract All".
  2. Choose the folder to extract to.
  3. Open the extracted folder and find the nanobot executable (`nanobot.exe`).
  4. Double-click to run.

- **macOS (.dmg)**:
  1. Open the `.dmg` file.
  2. Drag the nanobot app icon into the Applications folder.
  3. Eject the `.dmg` once copied.
  4. Open Applications and double-click nanobot to run.

- **macOS/Linux (.zip or .tar.gz)**:
  1. Extract the archive using built-in tools or command line.
  2. Open the extracted folder.
  3. For Linux, you might need to make the file executable. You can do this by right-clicking and selecting properties, then permissions, and enable "Allow executing file as program".
  4. Double-click or run the nanobot file from your terminal.

### Step 3: Running nanobot

After installing, start nanobot like any other app:

- On **Windows** or **macOS**, double-click the nanobot icon.
- On **Linux**, double-click or run the command `./nanobot` in the terminal inside the extracted folder.

When nanobot starts, you will see a simple window or command prompt asking for your input. You can type your requests or commands here. For example, say "Hello" or ask it to help organize a task.
